# controle-estacionamento
Restful api, Spring Boot e mysql para construção do projeto.

Este projeto visa criar uma aplicação que permita cadastrar veículos em vagas de estacionamento de apartamento.

Será usado o programa postman para apresentação dos métodos HTTP.
